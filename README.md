# Risk–Return Portfolio Analysis (Python)

## Overview
This project analyzes the risk–return characteristics of a portfolio consisting of four stocks using Python.
The objective is to evaluate expected returns, volatility, and to compare different portfolio investment strategies based on historical data.

## Data
- Asset prices: historical stock price data
- Time period: September 2023 - November 2025
- Source: CSV

## Methodology
- Risk and return estimation
- Correlation analysis
- Portfolio comparison
- Simple Strategy Assessment

## Results & Insights
- Investing Strategies relying on robustness & low risk can potentially outperform theoretically "optimal" when applied to new market conditions

## Limitations
- Assumptions of constant returns and no transaction costs
- Dividends are not taken into account
